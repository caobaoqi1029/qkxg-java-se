## Dev

1. 克隆本仓库 `git clone https://github.com/caobaoqi1029/qkxg-java-se.git --depth=1 && cd qkxg-java-se`
2. 使用 yarn 安装依赖 `yarn install`
3. 通过 gradle 构建 Java 

| 工具                              | 版本     |
| --------------------------------- | -------- |
| IntelliJ IDEA  (Ultimate Edition) | 2024.1.1 |
| yarn                              | 1.22.22  |
| node                              | 20.14.0  |
| gradle                            | 8.5      |
| Open JDK                          | 17.0.10  |
